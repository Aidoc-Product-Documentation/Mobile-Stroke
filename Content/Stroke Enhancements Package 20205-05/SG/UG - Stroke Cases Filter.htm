Customizing the Stroke Cases Filter

The stroke cases filter can be customized per site to capture additional cases in the feed.

Patient Imaging AI Flag Filter Configuration
Dedicated Stroke Imaging Irrelevant Standard
Non-dedicated CTA VO Standard
Non-dedicated CTA BA Standard configuration cases + cases with BA flag
Non-dedicated NCCT ICH Standard configuration cases + cases with ICH flag

It is possible to select one, both, or neither of the optional configurations for the dedicated stroke filter.

Applying Multiple Filters

Different filters work together in an AND configuration. Activating two different filters captures patient cards that correspond to both filters. E.g. when both Stroke Cases and Priority filters are enabled, the captured patient cards are stroke cases with Priority icons.
